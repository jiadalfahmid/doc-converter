Multi-File LaTeX Project Support:

To convert a full LaTeX project (including figures, bibliography, and multiple `.tex` files), please:

  1. Ensure your main LaTeX file (with `\begin{document}`) is named **`main.tex`**.
  2. Gather all `.tex` files, figures, and bibliography files into a single **`.zip`** archive.
  3. Upload the `.zip` file below.
